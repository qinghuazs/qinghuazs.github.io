02.SpringBoot系统要求

发表于 2019-06-18 更新于 2019-06-19 分类于 SpringBoot翻译 ， 2.1.5.RELEASE 阅读次数： 本文字数： 244

Spring Boot 2.1.5.RELEASE 需要 JDK8 并且兼容 JDK11。Spring Framework 5.1.7.RELEASE 或更高的版本。

为以下构建工具提供了明确的构建支持：

构建工具 版本
Maven 3.3+
Gradle 4.4+

Servlet 容器

Spring Boot 支持以下嵌入的Servlet容器：

容器 Servlet版本
Tomcat 9.0 4.0
Jetty 9.4 3.1
Undertow 2.0 4.0

您还可以将 Spring Boot 应用程序部署到任何兼容 Servlet 3.1以上的容器中。
